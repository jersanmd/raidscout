# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in RaidScout, please **do not** open a public issue.

Email: **security@raidscout.com**

We'll respond within 48 hours and work with you on a fix.

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest release | ✅ |
| Older releases | ❌ |

## Self-Hosted Instances

Security of self-hosted instances is the responsibility of the instance operator. Keep your Supabase project, environment variables, and API keys secure.

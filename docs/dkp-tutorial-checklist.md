# DKP & Character Claim Tutorial — Demo Checklist

## 🎬 Pre-Recording Setup

- [ ] Have a test server with DKP already enabled
- [ ] Have at least 2 test accounts (one staff, one regular member)
- [ ] Have a boss kill recorded so some members have DKP points
- [ ] Have an item marked for bid (or ready to mark one)
- [ ] Open two browser windows (incognito + normal) to simulate two users

## Part 8: Key Tips to Mention

- [ ] DKP is earned from boss kills based on the server's point rules
- [ ] Staff can manually adjust DKP with the ± button on any member
- [ ] Realtime notifications mean you never miss an outbid or win
- [ ] The bell icon shows all notification history
- [ ] DKP history is never deleted — resets create adjustments that zero out balances
- [ ] Guild-restricted auctions only show to members of that guild
